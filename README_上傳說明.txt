- sw.js

上傳到「圖示」資料夾覆蓋：

本版內容：
- 使用新版香豔臥房風 App icon
- sw.js 快取版本已更新為 royal-spring-sleep-pwa-v20
- 保留 v19 功能：
  - JSON 匯出修正，iPhone 優先使用分享選單，可選「儲存到檔案」
  - 月份背景圖功能
  - 月份背景固定修正
  - 全版面透明月曆
  - 日曆縮圖放大與向下堆疊
  - 侍寢狀態分為戴套、無套、留精、待寢
  - 修改日期、複製紀錄、點照片看原圖、備份文字等功能

若 iPhone 主畫面仍顯示舊圖示：
1. 等 GitHub Pages 更新完成後，先用 Safari 開網址重新整理。
2. 關閉主畫面 App 再重開。
3. 若仍是舊圖示，刪除主畫面圖示後重新加入主畫面。
4. 清除 Safari 網站資料前，請務必先匯出 JSON 或複製備份文字。
