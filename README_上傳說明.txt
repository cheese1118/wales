- index.html

本版內容：
- 使用新版「香豔古裝女子＋金色春字」icon
- icon 路徑改為根目錄
- sw.js 快取版本已更新為 royal-spring-sleep-pwa-v21
- 保留 v19 功能：
  - JSON 匯出修正，iPhone 優先使用分享選單，可選「儲存到檔案」
  - 月份背景圖功能
  - 月份背景固定修正
  - 全版面透明月曆
  - 日曆縮圖放大與向下堆疊
  - 侍寢狀態分為戴套、無套、留精、待寢
  - 修改日期、複製紀錄、點照片看原圖、備份文字等功能

如果 iPhone 主畫面 icon 沒變：
1. 先確認 GitHub Pages 網址重新整理後已載入新版。
2. 關掉主畫面 App 再重開。
3. 仍不變時，刪除主畫面舊圖示，再用 Safari 重新「加入主畫面」。
4. 若要清除 Safari 網站資料，請先匯出 JSON 備份。
